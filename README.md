# CSE-226 - Numerical Methods Sessional 

This repository contains my lab work for the **Numerical Methods Sessional** course (CSE226). 
I will be updating it regularly with the tasks and algorithm implementations completed in each lab session.

The purpose of this lab is to implement and explore various numerical methods for solving mathematical problems computationally.

## Methods Implemented
1. **Bisection Method** – Root-finding method using interval halving.  
2. **One-Point Iteration** – Iterative method for finding roots using \(x_{n+1} = g(x_n)\).  
3. **Newton-Raphson Method** – Uses derivatives to approximate roots: \(x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)}\).  
4. **Secant Method** – Iterative root-finding method using two initial approximations.  
5. **Gauss Elimination** – Direct method for solving systems of linear equations by reducing to row-echelon form.  
6. **Gauss-Jordan Elimination** – Extension of Gauss Elimination to reduced row-echelon form for direct solution.  
7. **Gauss-Seidel Method** – Iterative method for solving linear systems using the latest updated values.
